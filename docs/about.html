Project name

Disaggregated Memory Lab (DisaggMemLab) — an experimental framework and repository for exploring remote memory and networked storage approaches using RDMA, NVMe-over-Fabric and, conceptually, CXL.

Objectives

  • Provide reproducible scripts and CloudLab profiles to deploy experiments.
  • Measure and compare latency, throughput and CPU overhead of different transports.
  • Document practical steps to set up nvmet targets, initiators and measure performance.
  • Collect artifacts and troubleshooting tips for researchers and students.

Audience

Researchers, students and engineers interested in disaggregated memory, networked storage, and performance analysis on bare‑metal testbeds like CloudLab.
