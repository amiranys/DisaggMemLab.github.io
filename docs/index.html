Disaggregated Memory Lab (DisaggMemLab)

RDMA, NVMe‑oF and CXL experiments for CloudLab and bare‑metal labs

About this project

A compact lab project collecting documentation, CloudLab profiles and scripts to evaluate disaggregated memory-style setups using RDMA, NVMe-over-Fabric (NVMe-oF) and (conceptually) CXL. The canonical experiment uses two bare‑metal nodes (server and initiator) connected via an experiment network; the server exports a local NVMe namespace or exposes resources over the fabric while the client runs benchmarks (fio, perftest, STREAM).

This static site is a minimal project page you can publish to GitHub Pages. The full repository contains scripts in `scripts/`, CloudLab profiles in `cloudlab_profiles/`, and more advanced documentation in `docs/`.

Notes: CXL provides a different programming model (byte-addressable, coherent memory) compared to block storage over NVMe-oF. CXL hardware and firmware support is required; public CloudLab catalogs do not generally advertise CXL nodes — contact CloudLab staff if CXL is required for your experiments.
